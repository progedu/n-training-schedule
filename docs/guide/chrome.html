Google Chromeのインストール

Google Chromeのインストール方法を紹介していきます。

Google Chrome をインストールしよう

ニコニコ動画 や Google などのホームページを表示するのに使うのが ブラウザ です。

様々な種類のブラウザがありますが、本研修では Google Chrome（グーグル・クローム） を利用して説明をしていきますので、 Google Chrome のインストール方法を紹介します。もし、既に Google Chrome を利用している方はダウンロード・インストールの手順を読み飛ばし、次に進んでください。

Chrome のインストールと起動手順

Windows
Mac

お使いのブラウザを開きます。
URL（アドレス）を入力する欄に半角英数字で www.google.co.jp と入力して Enter を押し、検索エンジンの Google にアクセスしましょう。

Windows 10 の Edge の例

Edge でのURL入力欄

Internet Explorer の例

Internet Explorer でのURL入力欄

Google を開いたら、検索ボックスに Chrome と入力して Enter を押します。

Internet Explorer でのURL入力欄

検索結果から Chrome ダウンロードページへのリンクをクリックします。

ダウンロードページへのリンク https://www.google.co.jp/chrome/

Chrome のサイトが見つからない方は、こちらのリンクから Chrome のダウンロードサイトにアクセスしてください。

偽サイトに注意

「Chrome」で検索する人を狙った、悪質な偽サイトが確認されています。

偽サイトからソフトをダウンロードした場合、データが破損したり、個人情報が抜き取られたりする恐れがあります。

上の画像の赤枠で示しているように、本物のサイトの URL には「google.co.jp」あるいは「google.com」の文字列が入っています。

Chrome に限らず、ソフトウェアを検索してダウンロードする場合、開発元の公式サイトからダウンロードしようとしているかを確認しましょう。

ダウンロードボタンをクリックしてダウンロードを行います。

ダウンロードボタン

利用規約を良く読み、同意ができたらインストールボタンをクリックします。「使用統計データと……」のチェックは必要なければ外して構いません。

利用規約

次のような画面が表示されたら、「保存」をクリックしてください。 特に設定を変更していなければ、「ダウンロード」というフォルダに保存されているはずです。
もちろんここから実行しても構いませんが、ここではダウンロードフォルダを開いてみましょう。

ダウンロード処理の選択

「ダウンロードが完了しました」と表示されたら、「フォルダーを開く」をクリックします。

ダウンロード完了時の表示

ダウンロードフォルダを開くと、 ChromeSetup(.exe) というファイルがあるはずです。これをダブルクリックして起動します。

Chrome のインストーラーを起動する

セキュリティの警告が表示されたら「実行」をクリックします。

Chrome のインストーラーを起動する

ユーザーアカウント制御の画面が表示されたら、「はい」あるいは「実行」をクリックし、実行を許可します。

Chrome のインストーラーを起動する

しばらく待つと次のような画面となり、自動的に Chrome がダウンロード・インストールされます。

Chrome をインストール中の画面

インストールが終わると、自動的に Chrome が起動します。

Chrome を使い始める際に、便利にするための手順が表示されます。「開始する」をクリックします。

起動直後の画面

Google が提供するアプリをブックマーク（お気に入り）に登録する手順が表示されますが、特にこだわりがなければ「スキップ」をクリックしてください。

ブックマークのカスタマイズ画面

Chrome の背景デザインを選ぶ画面が表示されます。好きなものに変更できます。
特にこだわりがなければ「スキップ」をクリックしても構いません。

デザインのカスタマイズ画面

Chrome をデフォルトのブラウザ（Windows が標準で使うブラウザ）にするか尋ねる画面が表示されます。 Web プログラミングにおいては Chrome をデフォルトのブラウザに設定しておく方が便利なので、設定しておきましょう。「デフォルトとして設定」をクリックします。

既定のブラウザにするか尋ねる画面

すると、デフォルトのアプリを変更する Windows の設定画面が表示されます。
下の方に「Web ブラウザ」という欄があるので、ここに表示されている現在のブラウザ（Microsoft Edge や Internet Explorer）の名前をクリックし、表示されたアプリ一覧から Google Chrome をクリックします。

既定のアプリを設定する画面

変更してよいか聞かれるので、「強制的に変更する」をクリックします。
無事に設定ができたら、「デフォルトのアプリ」画面は閉じてください。

「強制的に変更する」をクリック

Chrome に戻ると、Google アカウントで Chrome にログインするかどうかを聞かれる画面が表示されます。特に必要性はないので、「いいえ」ボタンをクリックします。
もし Google のアカウント（ID）をお持ちであれば、ログインしても問題ありません。

Chrome にログインするか尋ねる画面

今後は、デスクトップに追加された「Google Chrome」のアイコンをダブルクリックすることで、Chrome を起動させることができます。

デスクトップから Chrome を起動できる

以上で、Google Chrome のインストールは完了です。

なお、インストールが正常に完了すれば、ダウンロードフォルダ内の ChromeSetup(.exe) は削除しても構いません。

お使いのブラウザを開きます。ここでは、 Safari を例にして説明していきます。
ブラウザには検索欄がありますので、 Chrome と入力して Enter を押します。

Chrome と入力し、 Enter キーを押す

Google の公式サイトから Chrome をダウンロードします。 https から始まる URL に google.co.jp と含まれているものが公式サイトです。

Chrome のダウンロードサイトにアクセスする
偽サイトに注意

「Chrome」で検索する人を狙った、悪質な偽サイトが確認されています。

偽サイトからソフトをダウンロードした場合、データが破損したり、個人情報が抜き取られたりする恐れがあります。

上の画像の赤枠で示しているように、本物のサイトの URL には「google.co.jp」あるいは「google.com」の文字列が入っています。

Chrome に限らず、ソフトウェアを検索してダウンロードする場合、開発元の公式サイトからダウンロードしようとしているかを確認しましょう。

画面中央に表示されている「Chrome をダウンロード」というボタンをクリックします。

ダウンロードボタンをクリック

利用規約に同意したら、「同意してインストール」をクリックします。

利用規約に同意したら「同意してインストール」をクリック

ダウンロードが完了したら、 Finder (ファインダー) を開きます。特に設定を変更していない macOS であれば、画面下部に位置する Dock の左端にあるはずです。

Finder を開く

ダウンロードフォルダを開き、先ほどダウンロードした googlechrome.dmg をダブルクリックで起動します。

googlechrome.dmg を開く

少し待つと、このような画面が表示されるので、Google Chrome.app と書いてあるアイコンを、下のフォルダのアイコンまでドラッグ&ドロップします。この操作によって、Google Chrome がお使いの Mac にコピーされます。

アイコンをドラッグ・アンド・ドロップ

コピーが完了したら、先ほどのアイコンが 2 つ並んだ画面を閉じてください。その後、Finder のデバイス欄に表示されている Google Chrome の横にある取り出しボタンをクリックしてください。なお、正常にインストールできていれば、先ほどダウンロードした googlechrome.dmg は削除しても構いません。

取り出しボタンをクリック

続いて、アプリケーションフォルダを開いて Google Chrome.app をダブルクリックしてください。

Google Chromeを起動

Google Chrome はインターネット上からダウンロードしたプログラムなので、 Mac がセキュリティ的に開いても問題ないかを聞いてきます。今回は Google の公式サイトからダウンロードしており、安全だと判断できるので「開く」をクリックします。

「開く」をクリック

「Google Chrome へようこそ」というダイアログが表示されたら、「Google Chrome を規定のブラウザとして設定する」をクリックしてチェックを入れます。
「使用統計データや……」という項目のチェックはよく分からなければ外して大丈夫です。
適切にチェックを入れ終えたら、「Google Chrome を起動」をクリックします。

「Google Chromeへようこそ」ダイアログ

「"Chrome"に変更」と書いてあるボタンをクリックします。

「"Chrome"に変更」をクリック

Chrome を使い始める際に、便利にするための手順が表示されます。「開始する」をクリックします。

起動直後の画面

Google が提供するアプリをブックマーク（お気に入り）に登録する手順が表示されますが、特にこだわりがなければ「スキップ」をクリックしてください。

ブックマークのカスタマイズ画面

Google アカウントで Chrome にログインするかどうかを聞かれる画面が表示されます。特に必要性はないので、「スキップ」ボタンをクリックします。
もし Google のアカウント（ID）をお持ちであれば、ログインしても問題ありません。

Chrome にログインするか尋ねる画面

無事、 Google Chrome が起動すればインストール完了です。

インストール完了

また、これから頻繁に Google Chrome を使っていくことになるので、 Dock に追加しておきましょう。 Dock 上で Chrome のアイコンを右クリック（MacBook, MacBook Air, MacBook Pro をお使いの場合、初期設定であれば 2 本指でクリック）し、「オプション」から「Dock に追加」をクリックします。

Dock に追加する

お疲れさまでした！
